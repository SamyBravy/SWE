		\textbf{Main Flow} & 
		\normallist{
			\item The model provider accesses the \texttt{Publish Model} page.
			\item The model provider enters the model name in the \texttt{Model Name} text box.
			\item The model provider enters the model description in the \texttt{Description} text box.
			\item The model provider enters the provider cost per token in the \texttt{Cost per Token Provider (€)} text box.
			\item The model provider presses \texttt{BROWSE FILES}.
			\item The system opens the file manager for selecting \textit{.safetensors} and \textit{.json} files.
			\item The model provider selects the files of the model and confirms.
			\item The system closes the file manager.
			\item The model provider presses \texttt{PUBLISH}.
			\item The system shows the message: \textit{"The model is waiting to be reviewed, it will be published after it has been validated"}.
		}

			\item Return to step 5.
